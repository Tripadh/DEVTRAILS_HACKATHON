# GigShield - Phase 1

GigShield is a parametric protection concept for gig delivery workers.
Payouts are triggered automatically by disruption signals such as rain, heat, and curfews.

## Live UI

After enabling GitHub Pages for this repository, open:

https://tripadh.github.io/DEVTRAILS_HACKATHON/

## Project Files

- `index.html` - Full designed project page (exact UI)
- `README.md` - Repository overview

## Enable GitHub Pages

1. Open repository Settings.
2. Go to Pages.
3. Under Build and deployment, set Source to Deploy from a branch.
4. Select branch `main` and folder `/ (root)`.
5. Save and wait 1-2 minutes.

Then refresh the Live UI link above.
